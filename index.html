Question:Palindrome Numbers

Q1.Write a program in c and in java to check whether a number is a palindrome of not.


A palindromic number or numeral palindrome is a number that remains the same when its digits are reversed. Like 16461, for example, it is "symmetrical".

Ans:In our program ,we are to check whether a number entered by user is a palindrome or not.
Hey coders!!we can perform it in a number of ways: In our approach, we are to proceed by initially storing the number in a variable and then finding it's reverse.
Later we can confirm it as palindrome by matching the original number with the reversed one.
and hence check out the following code!!

Code in C:

#include < stdio.h>
int main()
{
int n, reverse = 0, t;
printf("Enter a number to check if it is a palindrome or not\n");
scanf("%d", &n);
t = n;

while (t != 0)
{
reverse = reverse * 10;
reverse = reverse + t%10;
t = t/10;
}
if (n == reverse)
printf("%d is a palindrome number.\n", n);
else
printf("%d isn't a palindrome number.\n", n);

return 0;
}

Question:Factorial of a number

Q.3Write a program in c and in java to calculate the factorial of a number


For an integer n greater than or equal to 1, the factorial is the product of all integers less than or equal to n but greater than or equal to 1. The factorial value of 0 is defined as equal to 1.

Ans:Hey programmers!!roll up your sleeves to find out the solutions.
Inorder to find the factorial of a number, we find the product of all the natural numbers starting from 1 to the number itself.
And guess what!! The final obtained product is the factorial of the entered number.

Code in C:

#include < stdio.h>
int main()
{
int c, n, fact = 1;
printf("Enter a number to calculate its factorial\n");
scanf("%d", &n);
for (c = 1; c <= n; c++)
fact = fact * c;
printf("Factorial of %d = %d\n", n, fact);
return 0;
} Output

Code in JAVA:

Question:Bubble Sort

Q4.Write a program in c and in java to sort a given set of numbers in ascending order using bubble sorting technique.


Bubble Sort is the simplest sorting algorithm that works by repeatedly swapping the adjacent elements if they are in wrong order.

Ans: So, lets sort the code out lest time is shortened!!
Here, two consecutive elements of the array is checked, inside loop if in case of ascending order sorting the previous element is found to be larger than the later,they are swapped.
Nested loop is used to perform the operation throughout the array.

Code in C:

#include < stdio.h>
int main()
{
int array[100], n, c, d, swap;
printf("Enter number of elements\n");
scanf("%d", &n);
printf("Enter %d integers\n", n);
for (c = 0; c < n; c++)
scanf("%d", &array[c]);
for (c = 0 ; c < n - 1; c++)
{
for (d = 0 ; d < n - c - 1; d++)
{
if (array[d] > array[d+1])
{
swap = array[d];
array[d] = array[d+1];
array[d+1] = swap;
}
}
}
printf("Sorted list in ascending order:\n");
for (c = 0; c < n; c++)
printf("%d\n", array[c]);
return 0;
}
Output

public class BubbleSortExample {
static void bubbleSort(int[] arr) {
int n = arr.length;
int temp = 0;
for(int i=0; i < n; i++){
for(int j=1; j < (n-i); j++){
if(arr[j-1] > arr[j])

arr[j] = temp;

}
}
}

}
}
